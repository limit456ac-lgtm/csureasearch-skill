# CSUreasearch Skill

用于中文课程综述论文写作与 Word 文档修订，重点覆盖模板格式、文献证据管理、三线表与图表规范、GB/T 7714—2025 顺序编码制引用，以及最终排版检查。表格默认采用 1.5 磅顶线、0.75 磅表头分隔线和 1.5 磅底线。

## 安装

将仓库中的 `csureasearch` 文件夹复制到：

```text
~/.codex/skills/csureasearch
```

重新启动 Codex 或开启新会话，然后使用：

```text
$csureasearch
```

也可以让 Codex 的 `skill-installer` 从本仓库的 `csureasearch` 子目录安装。

## 目录

```text
csureasearch/
├── SKILL.md
└── agents/
    └── openai.yaml
```
